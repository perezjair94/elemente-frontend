## Before starting
PNPM (our recommendation due to its efficiency and speed)

 ```sh
  npm install -g pnpm
  ```

### Installation

Clone the repository

```sh
git clone https://github.com/perezjair94/elemente-frontend.git
```

Install the NPM packages

```sh
pnpm install
```

Run the project

```sh
pnpm run dev
```
